5. What is emmet? List some of the advantages emmet offers?

Ans: Emmet is a free plugin for text editors that allows us to write HTML, CSS, and other code formats faster by assisting through abbreviations that expand 
     into full code.
	
   Some of the advantages are-

	Saves a lot of time.

	Increase productivity.

	Efficient Programming.

	Allows to type shortcuts that are then expanded into full pieces of code after auto suggestive letter pops up.
