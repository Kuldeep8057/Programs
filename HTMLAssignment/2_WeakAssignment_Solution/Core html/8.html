HTML
HTML stands for HyperText Markup Language. It is used on the frontend and gives the structure to the webpage which you can style using CSS and make interactive using JavaScript.
CSS
CSS or Cascading Style Sheets is the language used to style the frontend of any website.
JavaScript
JavaScript allows you to add interactivity to your pages.
React.js
React is the most popular front-end JavaScript library for building user interfaces.
Node.js
Node.js is an open-source and cross-platform JavaScript runtime environment.
Express.js
A Express.js web framework used to build backend apps and APIs using JavaScript.
Mongodb
A NoSQL document database. Unlike a SQL database, it uses key-value pairs instead of columns and rows to store data.
